Oppgaver:

Oppgave 1:

Lag en loop som teller til 10

for (let i = 1; i <= 10; i++) {
    console.log(i);
}

Oppgave 2:

Lag en loop som kjører så lenge tallet i en variabel er over 0

let number = 5; // Du kan sette variabelen til ønsket startverdi
while (number > 0) {
    console.log(`Tallet er: ${number}`);
    number--; // Reduserer tallet med 1 hver iterasjon
}
